Гайд по использованию шаблона
=============================

Бойлерплейтом называют программный код, который вынужден повторяться из раза в раз от проекта к проекту. Такой код выносится отдельно и становится шаблоном, который можно переиспользовать.

Данный шаблон или бойлерплейт ориентирован на следующие аудитории:

- **Студенты**, желающие (или обязанные) создать из своей научной работы некоторый программный продукт, который смогут использовать другие люди, но не знающие, с чего начать.
- **Разработчики**, в рамках своей работы вынужденные создавать одни и те же пакеты/библиотеки и не желающие заниматься настройкой из раза в раз.

Особенности данного бойлерплейта:

- Настроенные зависимости для тестирования и сборки документации.
- Присутствие некоторого гайда по работе с шаблоном.
- Более легкий вкат в python вне google colab.
- Автоматическая сборка документации и тестирование с использованием workflows.
